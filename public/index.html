Extract CSS

Get all the CSS from a single webpage.

Example request urls:

  • https://www.google.com
  • https://www.projectwallace.com/~teamwallace/css-tricks/colors
  • https://cdn.jsdelivr.net/npm/tailwindcss/dist/tailwind.min.css

Please note that there is a 1 minute cache for each (normalized) url.

Source code available on GitHub.
